== README

1. bundle install
2. config .env
3. rake db:create
4. rake db:migrate
5. rake db:seed

Для каждой роли предусмотрен свой список кнопок, ссылок из меню.
Еще в зависимоти от роли виден свой список страниц.
Только админ может попасть во вкладку пользователи.

При желании можно добавить действия (изменить, удалить, добавить) для страниц и в зависимости от роли разрешать|запрещать определенные екшены.
